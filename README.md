# Airport Pickup & Delivery Scheduler

**Company:** ORBEM Solutions Private Limited

**Project Duration:** 01 June 2026 - 30 June 2026 (26 Working Days)

**Team Size:** 3 Students

## Project Overview

Airport Pickup & Delivery Scheduler is a smart working prototype that tracks pickup requests, delivery addresses, driver assignments, airport drop-offs, cargo handovers, and proof of delivery.

### What It Solves
- Tracks pickup request, delivery address, driver assignment, airport drop, cargo handover, and proof of delivery
- Coordinates first-mile and last-mile cargo movement around airport operations
- Replaces scattered WhatsApp messages, spreadsheets, and manual follow-ups with a centralized system

## Technology Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | HTML, CSS, JavaScript / React |
| **Backend** | Node.js Express or Python Flask |
| **Database** | MySQL / PostgreSQL / Firebase |
| **AI Layer** | Rule-based workflow logic |
| **Messaging** | WhatsApp Business API / Email API |
