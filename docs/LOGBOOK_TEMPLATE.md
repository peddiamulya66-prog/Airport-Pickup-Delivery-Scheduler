# How to Use This Template

1. **Copy this template** for each day of the internship
2. **Fill in details** at the end of each day
3. **Keep it updated** with real progress
4. **Be honest** about blockers and challenges
5. **Celebrate wins** by marking tasks as completed
6. **Push to GitHub** as part of documentation

## Tips

- **Be Specific:** Instead of "Worked on API", write "Built POST /api/requests endpoint with input validation"
- **Time Tracking:** Helps understand productivity and spot inefficiencies
- **Block Tracking:** Surface issues early so mentor can help
- **Evidence:** Reference code commits, pull requests, and documents
- **Consistency:** Updates should be daily, not weekly
